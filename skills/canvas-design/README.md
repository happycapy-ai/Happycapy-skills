# Canvas Design

Create beautiful visual art in PNG and PDF documents using design philosophy. Use this skill when you need to create posters, pieces of art, designs, or other static visual pieces. Creates original visual designs without copying existing artists' work to avoid copyright violations.

## Description

This skill helps you create museum-quality visual artwork through a two-step process:
1. Developing a design philosophy (aesthetic movement)
2. Expressing that philosophy visually on a canvas

The approach emphasizes visual expression through form, space, color, and composition with minimal text as visual accent.

## Use Cases

Use this skill when working on:
- Poster design
- Visual art creation
- Abstract compositions
- Design-forward PDF/PNG outputs
- Brand identity visuals
- Editorial design
- Exhibition materials

## Key Features

### Design Philosophy Creation
- Create visual philosophies/aesthetic movements
- Articulate design through space, form, color, composition
- Emphasize minimal text and visual-first communication
- Generate 4-6 paragraph design manifestos

### Canvas Creation
- Single-page, highly visual, design-forward outputs
- Museum or magazine quality work
- Sophisticated use of repeating patterns and perfect shapes
- Sparse, clinical typography with systematic reference markers
- Limited, intentional color palettes

### Philosophy Examples

The skill includes guidance for various aesthetic movements:
- **Concrete Poetry**: Communication through monumental form and bold geometry
- **Chromatic Language**: Color as the primary information system
- **Analog Meditation**: Quiet visual contemplation through texture and breathing room
- **Organic Systems**: Natural clustering and modular growth patterns
- **Geometric Silence**: Pure order and restraint

## Craftsmanship Standards

The skill emphasizes creating work that:
- Appears meticulously crafted with countless hours of labor
- Demonstrates master-level execution
- Shows expert-level attention to every detail
- Could be displayed in a museum or professional gallery
- Avoids AI-generated aesthetics through careful refinement

## Multi-Page Option

When requested, the skill can create multi-page documents that:
- Follow the same design philosophy across pages
- Provide distinct variations and creative twists
- Tell a story in a tasteful, visual way
- Resemble pages from a coffee table book
